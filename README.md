# Setup & Installation <br/>
### Clone the repo <br/>
git clone https://github.com/varada-angadi/portfolio.git <br/>

### Navigate into the folder <br/>
cd portfolio <br/>

### Install dependencies <br/>
npm install <br/>

### Run development server <br/>
npm run dev <br/>

### Build for production <br/>
npm run build <br/>
